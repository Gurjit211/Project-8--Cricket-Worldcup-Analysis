Project Overview

This project is a Power BI dashboard created to analyze Cricket World Cup match and player performance data.
The dashboard focuses on team results, batting and bowling statistics, match outcomes, and historical performance across countries.

The goal of this project is to provide clear insights into top-performing players, winning trends, and match results using interactive visuals.

What This Dashboard Shows

Total runs, balls faced, fours, and sixes by selected batsman

Total wickets, overs, maidens, and runs conceded by selected bowler

Player with most runs scored (Virat Kohli)

Player with most wickets taken (Lasith Malinga)

Match results summary (won, lost, tied, no result)

Country-wise match results (wins, losses, ties, no results)

Highest individual score by top 5 batsmen

Highest wickets taken in an innings by top 5 bowlers

Biggest winning margin in runs

Filters for batsman, bowler, country, and opposition

Key Insights

Virat Kohli has scored the highest total runs in World Cup matches

Lasith Malinga leads in total wickets taken

Some teams show consistently higher win counts across tournaments

A few batsmen have scored exceptionally high individual innings

Certain bowlers have achieved multiple 6+ wicket hauls in a single match

Match results are dominated by wins and losses, with very few ties

Visualizations Used

KPI cards for runs, wickets, and performance metrics

Line charts for highest individual scores and wickets taken

Donut charts for match result distribution

Tables for country-wise match outcomes

Interactive slicers for player, country, and opposition filtering
